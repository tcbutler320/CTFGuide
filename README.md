# The Capture the Flag Guide
The Capture the Flag Guide is the flagship project of CFTG.org. The focus of this project is to index resources on capture the flag events, techniques and tactics for solving CTF challenges, and to help beginners get started. The CTF Guide can be found at its [Wiki Page](https://github.com/tcbutler320/the-ctf-playbook/wiki)
